# Datathon Project – Malaysia Data Innovation Talent x DOSM 2025

## 📌 Overview

This project was developed as part of the Datathon competition, where the objective was to analyze datasets and generate actionable insights for decision-making. This project was selected as a **Top 10 Finalist at the state level**.

## 📊 Tools Used

* Microsoft Power BI
* Excel / Dataset preprocessing

## 📈 Project Objectives

* Analyze key trends from provided dataset
* Identify patterns and insights relevant to socioeconomic indicators
* Create an interactive dashboard for decision-makers

## 📊 Dashboard Features

* Interactive filters and slicers
* KPI tracking
* Trend analysis and comparisons
* Region/category-based insights

## 🔍 Key Insights

* **Addressing Structural Labor Market Challenges:** The project tackles the persistent issue of youth unemployment and skills mismatch in Malaysia, where a misalignment between education outputs and industry requirements has created significant regional disparities and high levels of skill-related underemployment.

* **Machine Learning-Driven Analysis:** Utilizing the CRISP-DM framework, the team implemented Random Forest machine learning models to analyze labor market datasets, which effectively identified complex patterns in unemployment trends and forecasted industry growth, surpassing the limitations of traditional, anecdotal career guidance methods.

* **Empowering Data-Informed Career Planning:** By transforming raw public data into actionable intelligence, the project provides a predictive "what-if" simulator and dashboard to help youths identify high-demand industries and align their skills with market needs, directly contributing to national objectives and SDGs 4 and 8.
